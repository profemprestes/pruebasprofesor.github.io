Ingeniería de Software

Desarrolla habilidades profesionales en desarrollo de software. Aprende metodologías ágiles, control de versiones y técnicas de testing para crear software de alta calidad.

Metodologías Ágiles

  • Scrum y Sprints
  • Kanban y flujo continuo
  • Roles y ceremonias
  • Gestión de backlog

Control de Versiones

  • Git fundamentals
  • Branching strategies
  • Colaboración en equipo
  • GitHub/GitLab workflow

Testing y QA

  • Unit testing
  • Integration testing
  • TDD methodology
  • QA best practices

Arquitectura de Software

  • Patrones de diseño
  • Clean Architecture
  • Microservicios
  • Escalabilidad
