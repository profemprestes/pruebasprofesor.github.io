Metodologías Ágiles

Las metodologías ágiles son enfoques iterativos e incrementales para la gestión de proyectos de software que priorizan la flexibilidad, la colaboración y la entrega continua de valor.

Framework Scrum

  • Roles: Scrum Master, Product Owner, Development Team
  • Eventos: Sprint Planning, Daily Scrum, Sprint Review, Sprint Retrospective
  • Artefactos: Product Backlog, Sprint Backlog, Incremento

Control de Versiones con Git

Git es un sistema de control de versiones distribuido que permite a los equipos colaborar eficientemente en el desarrollo de software.

Conceptos Fundamentales

  • Repositorios: Local y remoto
  • Commits: Guardado de cambios con mensajes descriptivos
  • Branches: Desarrollo paralelo de características
  • Merging: Integración de cambios entre ramas

Comandos Básicos de Git


# Iniciar repositorio
git init

# Añadir cambios
git add .
git commit -m "mensaje"

# Trabajar con ramas
git branch feature
git checkout feature
git merge feature

Comandos comunes de Git para el control de versiones.
